# TalentHQ HR Admin - 3-Minute Hackathon Pitch Script

**[0:00 - 0:30] Introduction (The Hook)**
"Hello Judges! Aaj hum present kar rahe hain **'TalentHQ'** – hamara next-gen HR Administration platform. Problem yeh hai ki traditional HR tools bahut cluttered aur outdated lagte hain, jisse macro-level decisions lene mein dikkat aati hai. Isliye, TalentHQ ko humne bilkul ek premium Enterprise SaaS ka look diya hai, with a highly professional, clean monochromatic blue theme."

**[0:30 - 1:15] The Executive Dashboard (Macro View)**
*(Screen: Show the main HR Admin Dashboard page)*
"Sabse pehle, yeh hai hamara Executive Dashboard. Ek HR Admin login karte hi seedha critical organization metrics dekhti hai. Yahan humne simply data dump nahi kiya hai—humne action-driven insights dikhaye hain. 
Aap dekh sakte hain Attrition Risk, Monthly Payroll Run Rate, aur Department breakdowns jise humne interactive *Recharts* ke through visualize kiya hai. Saath hi, yahan actionable dialogues hain for Pending Approvals aur Compliance Alerts, taaki kisi bhi urgent task miss na ho."

**[1:15 - 1:45] Employee Directory & Live Filtering**
*(Screen: Navigate to the Employee Directory and use the dropdown filters)*
"Let's move to the Employee Directory. Yahan humne live client-side filtering implement ki hai. Judges, if you notice, hum real-time mein Department ya Status ke basis pe immediately data filter kar sakte hain bina kisi lag ke. Everything is extremely snappy and intuitive."

**[1:45 - 2:30] AI-First Employee Profile (Micro View)**
*(Screen: Click into Sanya Sharma's Profile - Overview Tab)*
"Ab aate hain hamare sabse visually striking feature par: The details page. Kisi bhi employee pe click karte hi ek detailed overview open hota hai. Yeh ek standard list nahi hai; it's a dynamic data grid!
Isme hum employee ke immediate 'Team' members ko circular avatars mein dekh sakte hain. Neeche ek detailed 'Attendance Report' ka scatter plot hai. Aur right side pe ek unique **'AI Scoring'** module hai jo advanced soft skills—jaise Punctuality, Problem Solving, aur Adaptability—ko visually track karta hai. Ek HR ko basically kisi bhi employee ka 360-degree view ek hi modern screen pe mil jata hai."

**[2:30 - 3:00] Technical Scalability & Conclusion**
*(Screen: Stay on the profile or flip back to the stunning dashboard)*
"On the technical side, it's beautifully built. But the best part? Architecture sirf UI focus nahi hai, it's 100% production-ready. Hamare components ek `USE_MOCK_DATA` environment variable se wrapped hain. Abhi fast demo ke liye pure mock data pe local filtering ho rahi hai, but backend integrate karte hi, ek simple toggle change karne se yeh automatically real Axios GET requests par fallback kar jayega humare backend API ke liye.

To short mein, TalentHQ is visually outstanding, deeply analytical, and architected for real-world scaling. Thank you! We are open to questions."
